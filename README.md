# anime-face-detection
This is my first hand Deep learning project.
Having small dataset so I used pretrained Resnet50 model to train and predict for accuracy. It takes about 30-45s to detect whole image because I used Sliding window algorithm to detect each parts of the image.

Dataset: https://www.kaggle.com/thanhtungphannguyen/animeface

Pretrained-model: https://drive.google.com/open?id=1wb30fAqh1LF-bY4VD31E634vAlBTZnPj (>100MB)
